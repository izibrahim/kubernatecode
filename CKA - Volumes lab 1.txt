Create presistent volume:
==========================

apiVersion: v1
kind: PersistentVolume
metadata:
  name: pv-log
spec:
  capacity:
    storage: 100Mi
  accessModes:
    - ReadWriteMany
  persistentVolumeReclaimPolicy: Retain
  hostPath:
      path: /pv/log
	  

create the claim:
====================

apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: claim-log-1
spec:
  accessModes:
    - ReadWriteMany
  resources:
    requests:
      storage: 50Mi
	  
	  
Create the pod call the claim:
==============================

apiVersion: v1
kind: Pod
metadata:
  creationTimestamp: null
  labels:
    run: event-simulator
  name: event-simulator
spec:
  containers:
  - image: kodekloud/event-simulator
    name: event-simulator
    volumeMounts:
    - mountPath: /log
      name: webapp
  volumes:
    - name: webapp
      persistentVolumeClaim:
        claimName: claim-log-1
